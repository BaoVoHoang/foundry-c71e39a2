# BasicCalc

Open your browser. Do math. No setup, no dependencies.

## Features

- **Number Input & Display** — Click to build numbers. See your input and previous results at a glance.
- **Basic Operations** — Add, subtract, multiply, divide. The calculator stores your first number and waits for the second.
- **Calculate & Equals** — One click executes the operation and shows the result.
- **Clear Function** — Reset everything with a single button.

## Getting Started

### Install dependencies
```bash
npm install
```

### Run locally
```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).

### Build for production
```bash
npm run build
```

Output goes to the `dist/` directory.
